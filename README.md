# Инструкция по проверке сценария фильтрации
Для удобства проверки без необходимости устанавливать Unity, был сделан билд приложения, поэтому можно установить apk файл из [релиза](https://github.com/Mrfartam/InMoment/releases/tag/1.0) и запустить его на устройстве или эмуляторе (приложение поддерживает версии Android от 14 до 16). При первом запуске приложение потребует доступ к геопозиции для определения текущего местоположения пользователя, так что желательно, чтобы устройство было оснащено GPS-датчиком (при его отсутствии в качестве местоположения по умолчанию выставлен второй корпус ТГУ). Также необходимо, чтобы у устройства был доступ к интернету, так как происходит запрос к API для получения карты и списка заведений (проверка на отсутствие интернет соединения не реализована).

После запуска приложение сразу начинает искать заведения в радиусе 2км от местоположения пользователя, обычно это занимает не более 5 секунд.
Чтобы проверить выполнимость ключевого сценария, необходимо нажать (именно нажать, свайпы не работают) либо на стрелку на нижней панели, либо на поле ввода (поиск на момент реализации первого ключевого сценария не работает).  

Когда откроется меню со списком заведений, иконка приложения в левом верхнем углу панели заменится на кнопку фильтров. Необходимо будет нажать на неё для открытия окна фильтров. Далее – выбирайте любые из представленных там фильтров (кнопка сброса тоже работает), после выбора нажимайте "Готово". Если заведения будут найдены, в списке останутся те, которые удовлетворяют заданным фильтрам, а также на карте появятся метки этих заведений, иначе откроется меню с ошибкой, которое содержит кнопки смены фильтров и расширения области поиска:
- При нажатии на кнопку смены фильтров окно с ошибкой закроется и пользователю предоставится возможность изменить выбранные фильтры.  
- При нажатии на кнопку расширения области поиска, начнётся поиск ближайшего заведения, который на каждом шаге будет увеличивать радиус обнаружения на 0.5км до тех пор, пока заведение, удовлетворяющее выбранным фильтрам, не будет найдено.
